# Arduino BTK Course Repository

Bu depo, BTK Akademi'deki Arduino ve Tinkercad kursları kapsamında geliştirilen projeleri ve kaynak kodları içerir.

## İçerik Yapısı

Depo aşağıdaki üç ana modülden oluşmaktadır:

### 1. BTK Tinkercad ile Arduino (10 Saat)
Şu anda aktif olan ve proje dosyalarını içeren bölümdür. Aşağıdaki uygulamaları içerir:

| Proje Klasörü | Açıklama |
|---|---|
| **buzzer_ile_notalar** | Buzzer kullanarak nota çalma uygulaması |
| **buzzer_ile_rgb** | Buzzer ve RGB LED entegrasyonu |
| **dc_motor_kullanimi** | DC Motor kontrolü |
| **engelden_kacan_araba** | Engelden kaçan robot araba projesi |
| **esnek_sensor** | Esnek sensör kullanımı |
| **hirsiz_uyari_sistemi** | Hırsız alarm sistemi simülasyonu |
| **iki_buton_iki_led** | İki buton ile iki LED kontrolü |
| **karanlikta_yanan_led** | LDR ile karanlıkta yanan LED (Sokak lambası mantığı) |
| **kemer_uyari_sistemi** | Emniyet kemeri uyarı sistemi |
| **mesafe_sensor_uygulamasi** | Mesafe sensörü (HC-SR04) kullanımı |
| **potansiyometre_ile_servo_kontrol** | Potansiyometre ile servo motor açısı kontrolü |
| **potansiyometre_ile_isik_siddeti** | Potansiyometre ile ışık şiddeti ayarı |
| **rgb_buton** | Buton ile RGB LED renk değişimi |
| **röle** | Röle modülü kullanımı |
| **sicaklik_sensor_uygulama** | Sıcaklık sensörü (TMP36/LM35) uygulaması |
| **transistör** | Transistör kullanımı ve anahtarlama |
| **isik_sensor_deger_okuma** | Işık sensöründen analog değer okuma |
