# Katkıda Bulunma Rehberi

Projeye katkıda bulunmak istediğiniz için teşekkürler!

## Nasıl Katkıda Bulunabilirsiniz?

1.  **Hata Bildirimi:** Karşılaştığınız hataları Issues bölümünden bildirebilirsiniz.
2.  **Özellik İsteği:** Yeni projeler veya geliştirmeler önerebilirsiniz.
3.  **Pull Request:** Kendi geliştirdiğiniz projeleri veya düzeltmeleri gönderebilirsiniz.

### Pull Request Süreci

1.  Projeyi forklayın.
2.  Yeni bir branch oluşturun (`git checkout -b ozellik/yeni-ozellik`).
3.  Değişikliklerinizi yapın ve commit'leyin (`git commit -m 'Yeni özellik eklendi'`).
4.  Branch'inizi pushlayın (`git push origin ozellik/yeni-ozellik`).
5.  Bir Pull Request oluşturun.

## Kod Standartları

- Kodlarınızın anlaşılır ve yorum satırlarıyla açıklanmış olmasına özen gösterin.
- Türkçe değişken ve fonksiyon isimleri kullanılması tercih edilir (eğitim amaçlı olduğu için).
