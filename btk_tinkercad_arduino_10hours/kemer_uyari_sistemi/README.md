# Kemer Uyari Sistemi

Emniyet kemeri uyarı sistemi.

## Proje Hakkında
Bu proje, BTK Akademi kursu kapsamında geliştirilmiştir.

## Kurulum
1. `.ino` dosyasını Arduino IDE ile açın.
2. Gerekli devre bağlantılarını şemaya uygun olarak yapın.
3. Kodu kartınıza yükleyin.
